# Recordings

Access and manage your recorded video footage from cloud and local storage.

## Recording Types

### Continuous Recording

Records 24/7 regardless of motion activity.

- **Pros**: Never miss any event
- **Cons**: Uses more storage
- **Best for**: High-security areas

### Motion-Based Recording

Records only when motion is detected.

- **Pros**: Saves storage space
- **Cons**: May miss slow-moving events
- **Best for**: Low-traffic areas

### Scheduled Recording

Records during specified time windows.

- **Pros**: Customizable to your needs
- **Cons**: Requires planning
- **Best for**: Business hours monitoring

## Downloading Clips

### Download Single Clip

1. Navigate to the recording
2. Tap **Download** icon
3. Choose quality (Original/Compressed)
4. Clip saves to your device

### Create Custom Clip

1. Play the recording
2. Tap **‚úÇÔ∏è Clip** button
3. Set start and end points
4. Tap **Create Clip**
5. Choose to save or share

### Bulk Download

1. Go to **Recordings** ‚Üí **Manage**
2. Tap **Select**
3. Check multiple clips
4. Tap **Download Selected**

```{note}
Downloaded clips are saved to your Photos/Videos library on mobile or Downloads folder on desktop.
```

## Local Storage (NVR/DVR)

### Accessing Local Recordings

When connected to your local network:

1. Open **Recordings**
2. Select **Local Storage** tab
3. Browse recordings by date/camera

### HDD Management

1. Navigate to **Settings** ‚Üí **Storage** ‚Üí **Local**
2. View disk usage
3. Options available:
   - Format disk
   - Enable overwrite when full
   - Set recording schedule

```{warning}
Formatting will delete all existing recordings. Back up important footage first.
```

## Sharing Recordings

### Share via Link

1. Open the clip
2. Tap **Share** ‚Üí **Create Link**
3. Set expiration (1 day, 7 days, 30 days)
4. Copy or send link

### Direct Export

1. Open the clip  
2. Tap **Share** ‚Üí **Export**
3. Choose format (MP4, AVI)
4. Select destination app

### Email Evidence

1. Open the clip
2. Tap **Share** ‚Üí **Email**
3. Enter recipient
4. Clip attaches to email
