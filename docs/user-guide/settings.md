# Settings

Customize Night Owl Protect to match your preferences and requirements.

## Account Settings

### Profile Management

1. Go to **Settings** → **Account** → **Profile**
2. Manage:
   - Display name
   - Email address
   - Phone number
   - Profile photo

### Change Password

1. Go to **Settings** → **Account** → **Security**
2. Tap **Change Password**
3. Enter current password
4. Create new password
5. Confirm new password

### 2-Step Verification

Enhance security with additional authentication:

1. Go to **Settings** → **Account** → **Security**
2. Tap **2-Step Verification**
3. Choose method:
   - **Authenticator App** (most secure)
   - **SMS**
   - **Email**
4. Complete setup verification

```{important}
Save your backup codes securely. You'll need them if you lose access to your authentication method.
```

## Device Settings

### Camera Settings

For each camera, configure:

| Setting | Description |
|---------|-------------|
| Name | Custom name for the camera |
| Location | Room or area assignment |
| Video Quality | Resolution and frame rate |
| Night Vision | Auto/On/Off/Color |
| Audio | Enable/disable microphone |
| Status LED | Show/hide indicator light |
| Timestamp | Display date/time overlay |

### Recording Settings

- **Recording Mode**: Continuous, Motion, Scheduled
- **Pre-record Buffer**: 5-30 seconds
- **Post-record Time**: 10-60 seconds
- **Video Codec**: H.264/H.265

### Image Settings

Fine-tune video appearance:

- **Brightness**: -100 to +100
- **Contrast**: -100 to +100
- **Saturation**: -100 to +100
- **Sharpness**: 0 to 100
- **WDR**: Off/Low/Medium/High
- **Image Rotation**: 0°/90°/180°/270°

### Delete Account

```{warning}
Deleting your account will permanently remove all data, recordings, and settings. This action cannot be undone.
```

1. Go to **Settings** → **Account** → **Delete Account**
2. Enter password to confirm
3. Select reason (optional)
4. Confirm deletion
