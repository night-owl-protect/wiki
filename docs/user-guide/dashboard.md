# Dashboard Overview

The Dashboard is your central hub for monitoring all your security cameras and devices.

## Dashboard Layout

The main dashboard consists of several key areas:

### 1. Device Grid

The primary view showing all your connected cameras in a grid layout:

- **Single View** - View one camera at full size
- **2x2 Grid** - View 4 cameras simultaneously  
- **3x3 Grid** - View 9 cameras at once
- **4x4 Grid** - View up to 16 cameras

```{tip}
Tap any camera tile to expand it to full screen.
```

## Dashboard Features

### Device Groups

Organize your cameras by location or purpose:

1. Tap **Edit** in the top right
2. Select **Create Group**
3. Name your group (e.g., "Front Yard", "Office")
4. Drag cameras into the group
5. Tap **Done**

### Favorites

Mark frequently accessed cameras as favorites:

1. Long-press on a camera tile
2. Select **Add to Favorites**
3. Favorites appear first on your dashboard

### Quick Snapshots

Capture a still image from any camera:

1. Open the camera in live view
2. Tap the **üì∑ Camera** icon
3. Image saves to your device's photo library

## Customization Options

### Change Grid Layout

1. Go to **Settings** ‚Üí **Dashboard**
2. Select **Default Layout**
3. Choose your preferred grid size

### Enable Dark Mode

1. Go to **Settings** ‚Üí **Display**
2. Toggle **Dark Mode** on/off
3. Or select **Auto** to follow system settings

### Camera Order

Rearrange cameras on your dashboard:

1. Long-press on any camera tile
2. Drag to your preferred position
3. Release to drop
4. Changes save automatically

## Multi-Site Support

If you have cameras at multiple locations:

1. Tap the location name at the top
2. Select a different site from the dropdown
3. Dashboard updates to show that location's cameras

You can also view all sites simultaneously:

1. Go to **Settings** ‚Üí **Dashboard**
2. Enable **Show All Sites**
3. Cameras from all locations appear on one dashboard

## Dashboard Widgets

Add helpful widgets to your dashboard:

- **Weather** - Current conditions at camera location
- **Activity Summary** - Recent motion events
- **Storage Status** - Recording storage remaining
- **System Health** - Overall system status

To add widgets:

1. Scroll to the bottom of dashboard
2. Tap **Add Widget**
3. Select desired widget
4. Drag to reposition
