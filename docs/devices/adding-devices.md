# Adding Devices

Add new cameras and recorders to your Night Owl Protect account.

## Before You Begin

Ensure you have:

- ✅ Night Owl Protect app installed
- ✅ Account created and verified
- ✅ Device powered on
- ✅ Device connected to network (for NVR/DVR)
- ✅ WiFi password ready (for wireless cameras)

## Adding via QR Code (Recommended)

The fastest way to add a new device:

1. Open Night Owl Protect
2. Tap the **+** button
3. Select **Scan QR Code**
4. Point camera at QR code on device
5. Wait for connection (10-30 seconds)
6. Name your device
7. Assign to a room/location

```{tip}
The QR code is usually located on the device body, base, or in the quick start guide.
```

## Adding via Device ID

If QR scanning doesn't work:

1. Tap **+** → **Manual Setup**
2. Select device type:
   - Camera
   - NVR/DVR
   - Doorbell
3. Enter Device ID (found on device label)
4. Enter device password (default or custom)
5. Tap **Connect**

## Adding WiFi Cameras

### Smart Config Mode

1. Ensure camera is in setup mode (indicator blinking)
2. Open app → **+** → **Smart Config**
3. Enter your WiFi name and password
4. Wait for connection
5. Configure camera settings

### AP Mode (Direct Connection)

1. Put camera in AP mode (press reset 5 seconds)
2. Go to phone WiFi settings
3. Connect to camera hotspot (Night_Owl_XXXXX)
4. Return to Night Owl Protect
5. Tap **+** → **AP Mode Setup**
6. Enter your home WiFi credentials
7. Camera connects to your network

## Adding Wired Cameras

### PoE Cameras (NVR)

1. Connect camera to NVR's PoE port
2. Camera powers on automatically
3. Open Night Owl Protect
4. Go to your NVR
5. Camera appears in device list
6. Configure camera settings

### BNC Cameras (DVR)

1. Connect BNC cable from camera to DVR
2. Connect power adapter to camera
3. Camera appears on DVR channel
4. Access via Night Owl Protect

## Adding Additional Recorders

### New NVR/DVR

1. Complete initial recorder setup
2. Connect to internet
3. In Night Owl Protect, tap **+**
4. Scan QR code or enter manually
5. Enter admin password
6. All connected cameras appear automatically

## Sharing Device Access

### Invite Family Members

1. Go to device **Settings** → **Sharing**
2. Tap **Invite User**
3. Enter email address
4. Choose permission level:
   - **Admin** - Full control
   - **User** - View and control, no settings
   - **Viewer** - View only
5. Send invitation

### Accepting Invitations

1. Receive email invitation
2. Install Night Owl Protect (if needed)
3. Create account or sign in
4. Shared devices appear automatically

## Troubleshooting

### Camera Won't Connect

- Ensure camera is in setup mode (LED blinking)
- Check WiFi password is correct
- Verify 2.4GHz network (many cameras don't support 5GHz)
- Move closer to router
- Restart camera and try again

### QR Code Not Scanning

- Clean camera lens
- Improve lighting
- Hold phone steady
- Try entering Device ID manually

### Device Shows Offline

- Check power connection
- Verify network connectivity
- Restart device
- Check router firewall settings
