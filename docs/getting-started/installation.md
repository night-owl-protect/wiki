# Installation Guide

This guide will help you download and install Night Owl Protect on your device.

## Supported Platforms

| Platform | Minimum Version | Download |
|----------|-----------------|----------|
| iOS | 14.0+ | [App Store](https://apps.apple.com) |
| Android | 8.0+ | [Google Play](https://play.google.com) |
| Windows | 10+ | [Download](https://nightowlsp.com) |
| macOS | 11.0+ | [Download](https://nightowlsp.com) |
| Web | Modern Browser | [app.nightowlsp.com](https://app.nightowlsp.com) |

## Mobile App Installation

### iOS (iPhone/iPad)

1. Open the **App Store** on your iOS device
2. Search for **"Night Owl Protect"**
3. Tap **Get** to download and install
4. Once installed, tap **Open** to launch the app

### Android

1. Open **Google Play Store** on your Android device
2. Search for **"Night Owl Protect"**
3. Tap **Install**
4. After installation, tap **Open**

## Desktop Application

### Windows

1. Visit [nightowlsp.com/downloads](https://nightowlsp.com/downloads)
2. Click **Download for Windows**
3. Run the installer (`NightOwlProtect-Setup.exe`)
4. Follow the installation wizard
5. Launch from your Start Menu

### macOS

1. Visit [nightowlsp.com/downloads](https://nightowlsp.com/downloads)
2. Click **Download for macOS**
3. Open the downloaded `.dmg` file
4. Drag Night Owl Protect to your Applications folder
5. Launch from Applications or Spotlight

## Web Portal

Access Night Owl Protect directly from your browser:

1. Navigate to [app.nightowlsp.com](https://app.nightowlsp.com)
2. Use a modern browser (Chrome, Firefox, Safari, Edge)
3. Sign in with your Night Owl account

```{note}
The web portal offers full functionality without any installation required.
```

## Next Steps

After installation, proceed to the [Quick Start Guide](quick-start.md) to set up your account and connect your first device.
