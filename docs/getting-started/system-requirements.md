# System Requirements

Ensure your devices meet these minimum requirements for optimal performance.

## Mobile Devices

### iOS Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| iOS Version | 14.0 | 16.0+ |
| Device | iPhone 8 | iPhone 12+ |
| Storage | 200 MB | 500 MB |
| RAM | 2 GB | 4 GB+ |

### Android Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| Android Version | 8.0 (Oreo) | 12.0+ |
| Processor | Quad-core 1.5 GHz | Octa-core 2.0 GHz+ |
| Storage | 200 MB | 500 MB |
| RAM | 3 GB | 6 GB+ |

## Desktop Applications

### Windows

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| OS | Windows 10 | Windows 11 |
| Processor | Intel i3 / AMD Ryzen 3 | Intel i5 / AMD Ryzen 5+ |
| RAM | 4 GB | 8 GB+ |
| Storage | 500 MB | 1 GB |
| Graphics | DirectX 11 compatible | Dedicated GPU |

### macOS

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| OS | macOS 11 (Big Sur) | macOS 14+ |
| Processor | Intel Core i3 / M1 | M1 Pro/Max or Intel i5+ |
| RAM | 4 GB | 8 GB+ |
| Storage | 500 MB | 1 GB |

## Network Requirements

### Internet Speed

| Usage | Download | Upload |
|-------|----------|--------|
| Single camera (720p) | 2 Mbps | 1 Mbps |
| Single camera (1080p) | 5 Mbps | 2 Mbps |
| Multiple cameras (4x 1080p) | 20 Mbps | 8 Mbps |
| 4K streaming | 25 Mbps | 10 Mbps |

### Ports

The following ports must be open for proper functionality:

```
TCP: 80, 443, 8080
UDP: 1024-65535 (for video streaming)
```

```{note}
Most home routers automatically handle port forwarding via UPnP. If you experience connection issues, check your firewall settings.
```

## Web Browser Support

| Browser | Minimum Version |
|---------|-----------------|
| Google Chrome | 90+ |
